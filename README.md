## Quick Start

```bash
# Install dependencies
npm install

# Serve on localhost:3000
npm start

# Build for production
npm run build

# Format with prettier
npm run format

# Lint with eslint
npm run lint
```

## Deploy to Netlify

```bash
# Generate a production build and deploy it

  "scripts": {
    ...
    "predeploy": "npm run build",
    "deploy": "netlify deploy -p"
  },
```
